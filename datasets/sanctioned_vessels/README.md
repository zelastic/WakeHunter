# WakeHunter Sanctioned Vessels Index

A comprehensive dataset of **1,370 OFAC-sanctioned vessels** with detailed metadata for maritime sanctions compliance and risk management.

---

## Overview

The **WakeHunter Sanctioned Vessels Index** provides detailed information on vessels that have been sanctioned by the [Office of Foreign Assets Control (OFAC)](https://ofac.treasury.gov/).  
This dataset is essential for **maritime compliance**, **risk management**, and **sanctions screening** operations.

---

## Data Description

This index contains rich metadata on 1,370 sanctioned vessels including:

### Vessel Identifiers
- IMO numbers (99.5% coverage)
- MMSI numbers
- Call signs (56% coverage)
- Vessel names with normalized versions for search

### Vessel Information
- Vessel types (92.6% coverage) â€” including tankers, cargo vessels, fishing vessels, etc.
- Flag countries and country codes
- Tonnage measurements (gross tonnage and tonnage)

### Sanctions Details
- Sanctions programs (IRAN, DPRK4, RUSSIA-EO14024, etc.)
- Legal authorities
- Publication dates
- Sanctions types and lists
- Active status indicators

### Ownership Information
- Owner entities with relationship types
- Entity identifiers for cross-referencing

### Additional Data
- Vessel aliases and alternative names
- Former vessel flags
- Special features and remarks
- Additional identifiers
